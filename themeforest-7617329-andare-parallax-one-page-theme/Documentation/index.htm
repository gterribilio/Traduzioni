B) CSS Files and Structure - top

This theme uses thirteen CSS files. Five of them are the core structure, and the others contain plugin CSS.

----- Core CSS -----

- reset.css

This file resets all browsers to the same basic style. Things like margins and padding are all set to zero to start.

- base.css

This file holds all of the base elements of the theme. Such as links, typography, and other basic styles. Changes made to this document will effect the theme site wide. The file is clearly commented and structured for easy editing.

- layout.css

This file holds layout styles for the theme, and styles specific to sections. This file is the most extensive, and probably the first place to look if you plan to edit the theme css.

- grid.css

This file contains the core structure of the theme. It also handles the resizing of the layout based on the mobile device screen. The file utilizes media queries to accomplish this. No adjustments are necessary within this file.

- media-queries.css

This file handles some of the layout changes needed for mobile devices and smaller screens.

- colors/*.css

These files contain color schemes for the theme. The default color is red. So if you remove the link to any files within the "color" directory the theme will default to red.

- theme/*.css

These files are for the light or dark theme. If you remove the link to the files within the "theme" directory the theme will default to the light version.

- panel/options-panel.css

This css file is specific to the pull out options panel in the demo. You can remove this file for production.

----- Plugin CSS -----

-animate.min.css

This file contains the options for css animations. Check out http://daneden.github.io/animate.css/ for all the options.

- flexslider.css

This file contains the styles for the sliders found throughout the theme. Check out http://www.woothemes.com/flexslider/ for more info.

- prettyPhoto.css

This file contains the default theme for the PrettyPhoto plugin. See http://www.no-margin-for-errors.com/projects/prettyphoto-jquery-lightbox-clone/ for details on this plugin.

- font-awesome.css

This is the theme gets all of its icons from. Check out http://fortawesome.github.io/Font-Awesome/ for all the available icons.

- YTPlayer.css

This file contains the styles for the video backgrounds. See http://pupunzi.open-lab.com/mb-jquery-components/jquery-mb-ytplayer/ for all the options.

C) JavaScript - top

This theme imports between sixteen javascript files.

  1. jQuery - jQuery version 1.11.0
  2. jQuery.Appear - This plugin tracks an elements appearance in the browser viewport
  3. PrettyPhoto - This plugin allows for the elegant pop windows in the galleries.
  4. SmoothScroll - Makes same page links scroll smoothly.
  5. jQuery.Parallax - Creates the parallaz scrolling effect.
  6. jQuery Easing Plugin - This plugin is for fine tuning animation effects.
  7. SuperSlides - Used to create full screen slider.
  8. FlexSlider - This is the plugin that controls the slider animations.
  9. Isotope - Used to create and filter the portfolio layout.
  10. YTPlayer - Full screen video backgrounds.
  11. Fitvids - Fit a video within a container.
  12. CounTo - Used to create counters.
  13. Tipper - Used to create custom tooltips.
  14. Google Maps - Used to create the Google Maps background.
  15. Custom - This file contains some custom jQuery for use in the themea along with calls to plugins.
  16. Options Panel - This file contains some custom code for the options panel. (remove for production)
